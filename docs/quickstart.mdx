title: "Quickstart"
description: "Query your first Anarlog meeting from a terminal or agent."
---

This quickstart installs the current CLI from source and reads a meeting from the local Anarlog database.

## Prerequisites

- [Anarlog desktop](https://anarlog.so/download), opened at least once
- A recent Rust toolchain with `cargo`
- Git

<Steps>
  <Step title="Install from source">
    ```bash
    git clone https://github.com/fastrepl/anarlog.git
    cd anarlog
    cargo install --locked --path apps/cli
    ```
  </Step>
  <Step title="List recent meetings">
    ```bash
    anarlog --json meetings list --limit 5
    ```

    Copy the `id` of the meeting you want to inspect.
  </Step>
  <Step title="Read one meeting">
    ```bash
    anarlog --json meetings get MEETING_ID
    ```

    The result contains meeting metadata, the canonical note, generated summaries, participants, and action items.
  </Step>
  <Step title="Connect an agent">
    Configure an MCP client to launch `anarlog mcp`, or give the agent the [Anarlog skill](/agents/skills) so it can fall back to CLI JSON.
  </Step>
</Steps>

If the CLI cannot locate your database, open Anarlog once or follow [database troubleshooting](/troubleshooting#database-not-found).
