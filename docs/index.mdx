Anarlog is a local-first meeting notetaker. The desktop app records meetings and keeps notes, summaries, transcripts, participants, and action items on your computer.

The `anarlog` executable provides two read-only interfaces to that data:

- A CLI for people, scripts, and agents that can run shell commands.
- A local MCP server for agents that support the Model Context Protocol.

Both interfaces use the same application queries and never migrate or modify your database. CLI export can write a user-selected file, but it refuses to replace an existing file unless you pass `--force`.
